Working on Generative Models, Prompt Engineering & LLMs

This project is a personal initiative focused on exploring how Generative AI, Large Language Models (LLMs), and Prompt Engineering can be effectively integrated into each stage of the Software Development Life Cycle (SDLC).

Why This Project?

As AI systems become an integral part of modern software development, knowing how to ask the right questions is just as important as knowing how to write code. This website serves as a structured guide to designing high-quality prompts that improve clarity, productivity, and decision-making across SDLC phases.

What This Website Covers

  • Prompt templates mapped to SDLC stages
  • Use of LLMs for requirements analysis and design
  • Prompt-driven assistance for coding and testing
  • Best practices for reliable AI outputs
